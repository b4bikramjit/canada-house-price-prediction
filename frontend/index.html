About This App

Predict Canadian house prices using an ML model deployed with FastAPI. Enter property details to get an instant AI-generated estimate.

Model Highlights

  • CatBoost regression model
  • ~74% R2 score
  • Log-price training technique
  • Location frequency encoding
  • Amenities & property type features

Disclaimer

Estimates are approximate and not official appraisals. Actual market prices may vary based on factors not included in the dataset.
